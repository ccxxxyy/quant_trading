## 一、支持三类市场


| 市场 | 典型品种 | 数据来源 | 适合谁 |
|------|---------|---------|--------|
| **A 股** | 600519 茅台、000001 平安 | AkShare | 做股票的人 |
| **美股** | AAPL 苹果、TSLA 特斯拉 | yfinance | 做美股的人 |
| **中国期货** | au 黄金、rb 螺纹钢、IF 股指 | AkShare + CTP 网关 | **做期货的人** |

**期货用户推荐路径：**

1. **第 1 周**：用演示数据回测（不需要开户、不需要真实行情）
2. **第 2 周**：AkShare 拉取期货历史 K 线（如 `AU2412.SHFE`）
3. **第 3 周**：注册 [SimNow 模拟盘](https://www.simnow.com.cn)（免费），用 CTP 网关练模拟交易
4. **第 4 周及以后**：策略稳定后再考虑期货公司实盘（需自行开户）

> 注意：回测引擎对期货的**保证金、开平、今昨仓**做了简化模拟；CTP 实盘网关已实现框架，真实下单需 SimNow/期货公司账号。

## 二、流程（

### 第 0 步：确认环境

```powershell
cd D:\PythonProjects\quant_trading
uv sync --extra dev --extra web --extra data
uv run pytest tests/ -q
```

看到 `55 passed` 即环境 OK。

### 第 1 步：启动 Web 界面

```powershell
.\start_web.ps1
```

浏览器打开：**http://127.0.0.1:8888**

### 第 2 步：第一次回测

1. 左侧点 **「回测实验室」**
2. 策略选 **双均线**
3. **勾选「使用演示数据」**（不需要下载真实数据）
4. 初始资金改成你想要的数（如 `100000`）
5. 点 **「运行回测」**

你会看到：
- **权益曲线**：钱随时间怎么变
- **回撤曲线**：最坏时候亏多少
- **10 项指标**：收益率、夏普、最大回撤等
- **交易记录**：每一笔买卖

### 第 3 步：对比策略

回测页点 **「策略对比」**，系统同时跑 5 种策略，看谁历史表现最好。

### 第 4 步：命令行回测

新开一个终端（Web 那个别关）：

```powershell
cd D:\PythonProjects\quant_trading
uv run quant info
uv run quant backtest dual_ma --symbol DEMO.SSE --start 2024-01-01 --capital 100000
```

### 第 5 步：看策略说明

Web 左侧点 **「策略库」**，每张卡片是一种策略的说明和参数。

## 四、Web 界面九个页面怎么用

| 页面 | 干什么 | 新手建议 |
|------|--------|---------|
| **总览** | 4 个 KPI：数据量、策略数、默认资金、交易所 | 点卡片可跳转 |
| **数据管理** | 下载/查看行情 | 先勾选演示数据，有网再拉真实数据 |
| **回测实验室** | 选策略、设资金、看结果 | **最常用的页面** |
| **参数优化** | 选策略 → 自动网格搜索 → 最优参数排名 | 策略选定后用这个找最优参数 |
| **监控告警** | 查看告警记录、阈值配置、发送测试告警 | 了解系统的安全防线 |
| **模拟盘** | 虚拟资金下单交易、看持仓/挂单 | 回测满意后来这里"模拟实战" |
| **AI 实验室** | Alpha 因子列表、特征计算预览、ML 模型 | 进阶用户，了解 AI 预测原理 |
| **策略库** | 7 种策略的说明和参数 | 不懂策略先看这里 |
| **系统设置** | 风控阈值、默认配置 | 了解系统限制 |

## 五、七种策略怎么选

| 策略 | 一句话 | 你如果是… |
|------|--------|----------|
| 双均线 | 趋势跟着走 | 新手首选，最简单 |
| 布林带 | 价格偏离就反向做 | 觉得价格在区间内震荡 |
| RSI | 涨太多卖、跌太多买 | 喜欢抓反弹 |
| MACD | 金叉买死叉卖 | 做中线趋势 |
| 海龟 | 突破新高追、跌破止损 | **期货趋势党**常用 |
| 网格 | 区间内低买高卖 | 期货/币圈震荡市 |
| 配对 | 两只相关品种价差套利 | 进阶，需要两只标的 |

> 没有「最好」的策略，只有「适不适合当前行情」。用**策略对比**功能横向比较。

## 六、回测结果指标怎么读

| 指标 | 通俗意思 | 参考 |
|------|---------|------|
| 总收益率 | 一共赚/亏了多少 % | 正数赚，负数亏 |
| 年化收益率 | 折算成一年的收益率 | >10% 算不错（演示数据仅供参考） |
| 夏普比率 | 每承担 1 份风险赚多少 | >1 可用，>2 很好 |
| 最大回撤 | 从最高点最多跌了多少 | 越小越好，<-20% 要警惕 |
| 胜率 | 赢的交易占比 | 50% 左右正常 |
| 盈亏比 | 总盈利÷总亏损 | >1 说明赚的比亏的多 |
| 交易次数 | 一共买卖了多少次 | 太少可能样本不够 |

## 七、进阶：什么时候该用什么功能

```
刚入门 ──→ Web 演示回测 + 策略库
    │
    ├── 有明确品种 ──→ 下载真实数据 → 真实数据回测
    │
    ├── 策略选定了 ──→ Web「参数优化」页 → 自动网格搜索最优参数
    │
    ├── 想验证稳定性 ──→ Walk-Forward 验证
    │
    ├── 想模拟实战 ──→ Web「模拟盘」页 → 虚拟资金下单交易
    │
    ├── 想看 AI 预测 ──→ Web「AI 实验室」→ 计算特征 → 查看因子
    │
    ├── 想监控风险 ──→ Web「监控告警」页 → 查看阈值和告警记录
    │
    ├── 做期货 ──→ SimNow 模拟盘（CTP 网关）
    │
    ├── 做美股 ──→ IB 网关 + TWS
    │
    └── 想写自己的策略 ──→ 继承 BarSeriesStrategy → 放 strategies/ 目录
```

## 八、推荐日常命令（复制即用）

```powershell
# 进入项目（每次开终端先做这步）
cd D:\PythonProjects\quant_trading

# 启动 Web
.\start_web.ps1

# 查看系统信息
uv run quant info

# 演示回测（无需数据）
uv run quant backtest dual_ma --symbol DEMO.SSE --start 2024-01-01 --capital 100000

# 下载 A 股数据
uv run quant data fetch 600519.SSE --start 2023-01-01

# 下载期货数据
uv run quant data fetch AU2412.SHFE --start 2023-01-01

# 查看本地已有数据
uv run quant data list

# 跑全部测试
uv run pytest tests/ -v

# 多策略对比
uv run python scripts/demo_backtest.py
```
